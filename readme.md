## 具体要求：

单片机 AT89C52为控制核心，设计和实现了一个室内照明节能控制装置。系统采用传感技术探测光亮度和人体的存在，用单片机控制室内照明。AT89C52单片机作为该系统控制装置的智能部件，感知人体的存在则是用到了热释电红外传感器，以光敏元件为主要元件的电路用来检测光照强度。单片机根据取样电路采集到的光照强度、人体信号、时钟电路、时间等信息，控制照明电路的开关操作，从而实现照明节能控制。单片机具有手动控制
